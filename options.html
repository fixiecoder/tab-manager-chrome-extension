Title Patterns Color

Patterns:

  • example.com matches only example.com (any path)
  • *.example.com matches a.example.com, b.a.example.com (not example.com)
  • example.com/docs matches exactly the path /docs
  • example.com/docs/* matches any path under /docs/
  • *.example.com/* matches any path on any subdomain

Auto-close tabs (1s)

Add URL patterns. Matching tabs will auto close after 1 second.
